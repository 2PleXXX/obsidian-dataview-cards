# БЛОК 09 🧠 ЛОГИКА СЕКЦИЙ

> 💡 Этот блок определяет по какому полю из YAML карточки будут группироваться.

```js
sectionGroupField: "Section",
```

---

## 📌 Как это работает

- В YAML-файле у карточек должно быть поле `Section` (или другое, если вы поменяете  значение`sectionGroupField`).
- Проверяется наличие этого поля, и если оно есть, то оно может принимать значения (смотреть [[10 📦 СЕКЦИИ КАРТОЧЕК]])

## ✅ Пример YAML-записи

```yaml
Title: Azure Feedback
Section: // <- любое ваше наименование поля в YAML
  - SectionPoster // <- значения для этого поля
```
