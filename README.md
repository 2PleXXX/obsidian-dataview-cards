# Obsidian Dataview Enhanced

Расширенный DataviewJS-скрипт для Obsidian.

## Установка

1. Скачайте `dataview-config.md` и `dataview-logic.js` из этого репозитория.
2. Поместите оба файла в вашу Obsidian vault.
3. Убедитесь, что Dataview включен и поддерживает JS.

## Обновления

Скрипт автоматически проверяет наличие новой версии при каждом запуске.
